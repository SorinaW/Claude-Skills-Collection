# Doc Coauthoring Skill Dependencies

## Type
Guidance-only skill (no installation required)

## What it provides
Structured 3-stage workflow for collaborative document creation:
1. **Context Gathering** - Close knowledge gaps with clarifying questions
2. **Refinement & Structure** - Build sections iteratively (brainstorm, curate, draft)
3. **Reader Testing** - Test with fresh Claude to catch blind spots

## Use cases
- Technical specs
- Decision docs
- PRDs
- RFCs
- Proposals
