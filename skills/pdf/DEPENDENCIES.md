# PDF Skill Dependencies

## Python libraries
```bash
pip install pypdf pdfplumber reportlab
```

## Optional (for OCR on scanned PDFs)
```bash
pip install pytesseract pdf2image
```

## System tools (optional)
- **poppler-utils** - pdftotext, pdfimages
- **qpdf** - merge, split, rotate PDFs
- **pdftk** - alternative PDF toolkit
