# Internal Comms Skill Dependencies

## Type
Guidance-only skill (no installation required)

## What it provides
Templates and guidelines for internal communications:
- 3P updates (Progress, Plans, Problems)
- Company newsletters
- FAQ responses
- Status reports
- Leadership updates
- Incident reports

## Files included
- `examples/3p-updates.md`
- `examples/company-newsletter.md`
- `examples/faq-answers.md`
- `examples/general-comms.md`
