## Skill Architecture Patterns

### 1. Standard SKILL.md Structure
```markdown
---
name: skill-name
description: What this skill does
---
[Instructions for Claude]
```

### 2. Progressive Disclosure (500-line rule)
- Main SKILL.md under 500 lines
- Separate /resources folder for details
- Claude loads incrementally

### 3. Auto-Activation (diet103 pattern)
- Hooks trigger skills automatically
- skill-rules.json defines patterns
- No manual selection needed
