# Skills Research Collection

This is a research repository for Claude skills discovered across GitHub. Use this to:
- Browse skill candidates
- Review before improving and adding to your collection
- Track which skills you've evaluated

## Quick Links

- [Anthropic Official Skills](./resources/anthropic-skills.md)
- [Community Collections](./resources/community-collections.md)
- [Top 50 Skills Analysis](./resources/top-50-analysis.md)
- [Skills to Review](./resources/review-queue.md)

## How to Use

1. Browse the resources folder for skill links
2. Tell Claude which skill you want to review
3. Claude fetches and shows you the original
4. You decide: improve and add, or skip
5. Mark as reviewed in review-queue.md
