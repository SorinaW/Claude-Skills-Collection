# Brand Guidelines Skill Dependencies

## Type
Guidance-only skill (no installation required)

## What it provides
Anthropic's official brand colors and typography for consistent visual identity.

## Brand assets included
- **Colors**: Dark (#141413), Light (#faf9f5), Accents (orange, blue, green)
- **Typography**: Poppins (headings), Lora (body)

## Optional fonts
For best results, install:
- Poppins font
- Lora font
(Falls back to Arial/Georgia if unavailable)
